# Manipulação e Preparação de Dados para Inteligência Artificial

Este projeto apresenta uma introdução prática à **manipulação e preparação de dados para modelos de Inteligência Artificial** utilizando **Python**, **Pandas** e **Scikit-learn**. O foco é demonstrar as principais etapas de limpeza, exploração e preparação dos dados antes do treinamento de um modelo de Machine Learning.

O material foi desenvolvido para fins didáticos, auxiliando estudantes na compreensão das etapas fundamentais do pré-processamento de dados.

---

## Objetivos

- Aprender a manipular dados utilizando Pandas;
- Explorar informações estatísticas de um conjunto de dados;
- Identificar e tratar valores faltantes;
- Filtrar e transformar dados;
- Criar novas variáveis;
- Construir gráficos simples para análise exploratória;
- Preparar os dados para treinamento de modelos de IA;
- Treinar um modelo básico de Regressão Linear.

---

## Funcionalidades Desenvolvidas

Durante a aula são apresentados diversos recursos da biblioteca Pandas, incluindo:

- Criação de DataFrames;
- Exploração de dados (`info()` e `describe()`);
- Identificação de valores nulos;
- Tratamento de dados ausentes utilizando médias;
- Filtragem de registros;
- Criação de colunas derivadas;
- Construção de gráficos de barras;
- Seleção de variáveis de entrada e saída para Machine Learning.

---

## Modelo de Machine Learning

Após a preparação dos dados, é utilizado um modelo de **Regressão Linear** para realizar previsões.

As etapas incluem:

- Separação entre variáveis independentes (X) e variável alvo (y);
- Divisão dos dados em treino e teste;
- Treinamento do modelo;
- Geração de previsões para os dados de teste.
